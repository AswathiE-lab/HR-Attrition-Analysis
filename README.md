# HR Attrition Analysis

## Objective
To analyze employee attrition patterns using the IBM HR Analytics dataset and identify key factors influencing turnover.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
IBM HR Analytics Employee Attrition & Performance Dataset

## Key Insights
- Attrition rate is approximately 16%.
- Lower job satisfaction correlates with higher attrition.
- Employees with lower monthly income tend to leave more frequently.
- Work-life balance impacts retention.
- Certain departments show higher turnover.

## Business Recommendations
- Improve employee engagement initiatives
- Offer competitive compensation
- Enhance work-life balance policies
- Implement targeted retention strategies
